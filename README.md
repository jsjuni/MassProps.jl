# MassProps.jl

[![Build Status](https://github.com/jsjuni/MassProps.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/jsjuni/MassProps.jl/actions/workflows/CI.yml?query=branch%3Amain)

MassProps.jl is an expermental Julia port of the R package [massProps](https://github.com/jsjuni/massProps).

Recursively calculates mass properties (mass, center of mass,  moments and products of inertia, and optionally, their uncertainties) for arbitrary decomposition trees. R. L. Zimmerman, J. H. Nakai. (2005) <https://www.sawe.org/product/paper-3360/>).
